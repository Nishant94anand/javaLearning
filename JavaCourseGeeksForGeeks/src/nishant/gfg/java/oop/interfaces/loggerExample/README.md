# Logger Interface Example

This example demonstrates advanced capabilities of Java interfaces.

## Concepts Covered

- Static methods in interfaces
- Main method inside interface
- Interface as a utility holder

## Key Idea

Interfaces in Java (post Java 8) are not limited to just abstract contracts.
They can also contain static methods and even a main method.

## Highlights

- No class required to run the program
- Direct execution via interface main method
- Utility-style design using static methods

## Takeaway

Interfaces can act as:
- Behavior contracts
- Utility containers
- Lightweight execution units (for testing/demo)
